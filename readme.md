## Install

using npm:

```bash
$ npm install --global pubchat-cli
```
using npx: 

```bash
$ npx pubchat-cli
```


## CLI

```
$ pubchat-cli --help

  Usage:
    $ pubchat-cli 
	
  Example:
    $ pubchat-cli
      username: (put your username)
      channelName: (put the channelName you wanna join)

[enjoy chatting like a hacker :D]

```
